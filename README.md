# Language Tutor with Langflow

An AI-powered language tutor built with Langflow.

## What it does
- Upload a CSV file with vocabulary words in any language
- AI detects the language automatically
- Generates a short story using only your vocabulary words
- Provides English translation
- Asks comprehension questions

## Languages Supported
- Spanish, Hindi, French, German, Japanese, Portuguese and more!

## How to use
1. Install Langflow
2. Import language_tutor.json into Langflow
3. Upload any vocabulary CSV file
4. Give a theme and get your story!

## Built with
- Langflow
- OpenRouter API
